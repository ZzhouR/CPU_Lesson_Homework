步骤1：创建虚拟环境

# 1. 创建名为heart_sound（可自拟名称）的虚拟环境，指定python版本为3.8（兼容性较好）
conda create -n heart_sound python=3.8 -y

# 2.  激活虚拟环境
conda activate heart_sound

步骤2：安装必要库

# 1. 首先安装基础数值计算库
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple numpy==1.21.6 scipy==1.7.3

# 2. 安装核心数据科学库
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pandas==1.3.5

# 3. 安装matplotlib（指定稳定版本）
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple matplotlib==3.5.2

# 4. 安装librosa及其依赖
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple numba==0.55.2 llvmlite==0.38.1
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple librosa==0.9.2
pip install tqdm
pip install PyWavelets
pip install antropy -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple xgboost

步骤3：将虚拟环境添加到Jupyter中

# 1. 安装ipykernel
pip install ipykernel

# 2. 将当前虚拟环境添加到Jupyter
python -m ipykernel install --user --name heart_sound --display-name "Python (heart_sound)"


注：以上3个步骤基本在3-5分钟即可完成。

步骤4：启动Jupyter Notebook


步骤5：在Jupyter中使用虚拟环境
打开Jupyter，打开主程序：Chapter4_heartsound_main，点击菜单栏Kernel->Change kernel->选择Python (heart_sound)，即可运行代码
